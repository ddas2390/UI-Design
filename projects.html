Project 4: Solitaire Puzzle Board

Project – Solitaire Puzzle Board: This project involved implementing a recursive backtracking algorithm in Java, which finds a solution to the solitaire puzzle. The solution was to be stored as a sequence of boards: one for each move. The board was to be implemented as a Java class with an internal 7x7 matrix (two dimensional array). In the backtracking algorithm all possible jumps for a given situation had to be computed using methods and constructors.
